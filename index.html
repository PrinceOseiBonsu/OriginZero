How It Evolved
01
Crawl Wikipedia
Live API pulls articles and links connected to the topic — seeding the knowledge tree
02
Build Graph
NetworkX constructs a directed graph of 60+ nodes and 474 edges from real Wikipedia data
03
Run PageRank
Algorithm scores every node by influence — separating true root causes from related topics
04
AI Narrative
Groq llama3 reads the ranked chain and generates a plain English origin story in seconds
